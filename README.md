# GDCelestial

Celestial physics implementation for Godot 4.

## Currently supported

- [x] 2D conic sections.
- [ ] 3D conic sections.
- [ ] 2D N-body.
- [ ] 3D N-body.

## Additional information

Based on [gdextension template](https://github.com/nathanfranke/gdextension) by Aaron Franke and Nathan Franke.
